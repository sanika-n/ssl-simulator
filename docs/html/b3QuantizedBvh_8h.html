B3_ATTRIBUTE_ALIGNED16(struct) b3QuantizedBvhNode B3_ATTRIBUTE_ALIGNED16 ( struct  )

b3QuantizedBvhNode is a compressed aabb node, 16 bytes. Node can be used for leafnode or internal node. Leafnodes can point to 32-bit triangle index (non-negative range). b3OptimizedBvhNode contains both internal and leaf node information. Total node size is 44 bytes / node. You can use the compressed version of 16 bytes.
