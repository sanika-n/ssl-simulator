Detailed Description

template<typename T, unsigned Alignment>
class b3AlignedAllocator< T, Alignment >

The b3AlignedAllocator is a portable class for aligned memory allocations. Default implementations for unaligned and aligned allocations can be overridden by a custom allocator using b3AlignedAllocSetCustom and b3AlignedAllocSetCustomAligned.
