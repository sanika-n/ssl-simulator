B3_ATTRIBUTE_ALIGNED16 ( class  )

b3Vector3 can be used to represent 3D points and vectors. It has an un-used w component to suit 16-byte alignment when b3Vector3 is stored in containers. This extra component can be used by derived classes (Quaternion?) or by user Ideally, this class should be replaced by a platform optimized SIMD version that keeps the data in registers
