Detailed Description

InitPoseArgs is mainly to initialize (teleport) the robot in a particular position No motors or controls are needed to initialize the pose. It is similar to moving a robot to a starting place, while it is switched off. It is only called at the start of a robot control session. All velocities and control forces are cleared to zero.
